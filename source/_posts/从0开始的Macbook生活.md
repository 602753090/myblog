## 验货篇
接下来收到Macbook最重要的第一步就是**验货**啦！
### 收货开箱
#### 录视频
- 开箱过程中一定要全程录制视频！检查箱子有无磕碰，确认无误后开箱！

- 拆快递包装后检查**纸箱包装**是否完好！确认无误后开启！

- 检查**Mac包装**是否完好！确认无误后拆封！

- 检查**Mac外观**，有无磕碰划痕
#### 序列号
核对纸箱包装序列号、Mac包装序列号、Mac底部序列号、售后卡序列号是否一致！
### 开机
检查完毕后**开机**！

全部设置完成后，进入**桌面**，点击左上角**苹果图标**，关于本机，查看**序列号**是否一致！

## 使用篇
### 触控板
Macbook的触控板用了都说好！那么到底怎么用呢？

- 轻按 = 鼠标左键点击
- 重按可以触发特定操作（感觉用的不多）
- 双指轻按 = 鼠标右键点击
- 双指轻扫 = 滚动屏幕
- 双指捏合 = 放大/缩小
- 双指从右侧边缘划入 = 显示小组件
- 三指上滑 = 显示所有打开的窗口
- 三指下滑 = 当前app相关窗口（有时候一个app会打开多个窗口）
- 三指左右轻扫 = 切换全屏窗口（就是在所有全屏的窗口之间切换，没有全屏的窗口不会切到）（一般把主要使用的app全屏，然后使用这个手势就可以在全屏app以及桌面之间切换，灰常方便）
- 四指捏合 = 显示/关闭**全部app**（类似ipad和iphone的桌面，包含了电脑上的全部app，但是感觉不太好用，所以我自己设置了快捷键，见下文）

就是酱紫！快试一试吧！刚开始懵懵的，木有关系！熟悉了就会发现真是***太妙了***！

### 键盘
键盘最主要就是一些快捷键啦！多试几次，用习惯就好咯！
#### 主要按键
这里主要介绍一些和win不一样的按键！

- 最上面一排 - 按下实现图标上的功能（F4的搜索功能有时候很方便，可以全局搜索）
- command键 - 大致相当于win的control键，win里使用control键的快捷键一般把control换成command就可以在mac上用啦！
- option键 - 还没太搞懂，大概就是可以实现额外功能，按住这个键可以显示更多的选项
- 左下角fn键（地球仪） - 默认用来输入**表情**
- 中英切换键 - 按一下切换输入法，**长按**亮绿灯大写锁定
- delete键 - 就是退格键，要实现delete的功能需要同时按fn

#### 主要快捷键
这里介绍一些比较实用的快捷键！

- 截图 - command + shift + 5
- 复制/粘贴 - command + C/V
- 退出当前app - command + Q
- 关闭当前窗口 - command + W
- 打开全部app - 这个没有默认快捷键，我为了方便设置了command + L，到设置里**最下面-键盘-快捷键**设置就好！
